# Radio

Simple radio field with optional label & description, rendered in *Formfield* component.

## Usage

#### Default

<Playground>
  {() => {
    const [checked, setChecked] = React.useState(false);
    return (
      <Radio
        id="some_check"
        checked={checked}
        onChange={() => {setChecked(!checked)}}
      />
    );
  }}
</Playground>
