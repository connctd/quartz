# Toast

Dynamic Toast Notification component to show action feedback or error messages to the user.

## Usage

#### Title

<Playground>
  <Toast
    title="I'm a Toast Notification!"
  />
</Playground>

#### Description

<Playground>
  <Toast
    title="I'm a Toast Notification!"
    description="And I have something important to say..."
  />
</Playground>

#### No Icon

<Playground>
  <Toast
    title="I'm a Toast Notification!"
    description="And I have something important to say..."
    noIcon
  />
</Playground>

#### Custom Icon

<Playground>
  <Toast
    title="I'm a Toast Notification!"
    description="And I have something important to say..."
    icon={EmoticonCoolOutlineIcon}
  />
</Playground>

#### Closable

<Playground>
  <Toast
    title="I'm a Toast Notification!"
    description="And I have something important to say..."
    closable
    onClose={() => { /* Do stuff... */ }}
  />
</Playground>

#### Custom Width

<Playground>
  <Toast
    title="I'm a Toast Notification!"
    description="And I have something important to say..."
    width="80%"
  />
</Playground>

#### Single Action

<Playground>
  <Toast
    title="I'm a Toast Notification!"
    description="And I have something important to say..."
    actions={[{label: 'Action', onClick: () => { /* Do stuff */ }}]}
  />
</Playground>

#### Multiple Actions

<Playground>
  <Toast
    title="I'm a Toast Notification!"

    actions={[
      {label: 'Cancel', onClick() { /* Do stuff */ }, secondary: true},
      {label: 'Confirm', onClick() { /* Do stuff */ }}
    ]}
  />
</Playground>
