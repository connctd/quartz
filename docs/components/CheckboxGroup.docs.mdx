# Checkbox Group

Group checkboxes together for a specific form section. Requires a label and can have a description.

## Usage

#### Label

<Playground>
  <CheckboxGroup label="Your favourite philosopher">
      <Checkbox
        id="nietzsche"
        checked={true}
        onChange={() => {
          // do stuff
        }}
      >
        Friedrich Nietzsche
      </Checkbox>
      <Checkbox
        id="confucius"
        checked={true}
        onChange={() => {
          // do stuff
        }}
      >
        Confucius
      </Checkbox>
      <Checkbox
        id="socrates"
        checked={true}
        onChange={() => {
          // do stuff
        }}
      >
        Socrates
      </Checkbox>
      <Checkbox
        id="plato"
        checked={false}
        onChange={() => {
          // do stuff
        }}
      >
        Plato
      </Checkbox>
    </CheckboxGroup>
</Playground>
