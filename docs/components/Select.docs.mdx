# Select

Select input element to use inside forms or standalone. Optional label, description & error. Rendered in *Formfield* component.

## Usage

#### Placeholder

<Playground>
  <Select
      placeholder="Right hand, or left hand?"
      onChange={(event) => {
        // do stuff
      }}
    >
      <Select.Option value="left">Left</Select.Option>
      <Select.Option value="right">Right</Select.Option>
    </Select>
</Playground>

<Playground>
