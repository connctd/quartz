# Input

Input element to use inside forms or standalone. Optional label, description, error & prefix. Rendered in *Formfield* component.

## Usage

#### Placeholder

<Playground>
  <Input
      placeholder="What's your name?"
      onChange={(event) => { /* do stuff */ }}
    />
</Playground>

#### Label

<Playground>
  <Input
      label="Name"

    />
</Playground>
