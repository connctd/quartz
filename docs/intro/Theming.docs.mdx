#### Customize

To customize the theme, you need to pass your theme to `ThemeProvider`. be careful to include all properties you can see in the `defaultTheme` above.
The `ThemeProvider` needs to wrap all Quartz components, so ideally put it next to your apps root.

<Code className="language-jsx">{`
import { ThemeProvider } from 'emotion-theming';
import { defaultTheme } from '@connctd/quartz';
\n
const myTheme = {
  ...defaultTheme,
  primary: '#0099ff'
};
\n
<ThemeProvider theme={myTheme}>
  { /* your app goes here */ }
</ThemeProvider>
`}</Code>
